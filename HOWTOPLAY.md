# Colony Sim — Как играть

## Запуск

```bash
cd game
npm install
npm run dev
```

Откроется `http://localhost:3000`.

## Управление

| Действие | Как |
|---|---|
| **Двигать поселенца** | Клик по проходимой клетке (трава, песок, грунт, камень) |
| **Собрать ресурс** | Клик по красному квадрату (дерево/камень) |
| **Построить здание** | Кнопка здания внизу → клик по пустой клетке |
| **Отменить строительство** | Кнопка `[CANCEL]` |
| **Сохранить** | Кнопка `[SAVE]` |
| **Загрузить** | Кнопка `[LOAD]` |
| **Очистить сохранение** | Кнопка `[CLEAR]` |

## Строительство

### Как строить

1. Сначала **собери ресурсы** — кликай по красным квадратам (wood, stone)
2. Нажми кнопку здания в нижней панели (например, `[House] wood:8 stone:3`)
3. Кликни по **пустой проходимой клетке** на карте
4. Поселенец автоматически пойдёт и построит здание

### Типы зданий

| Здание | Ресурсы | Время | HP |
|---|---|---|---|
| **House** | wood:8, stone:3 | 10 тиков | 100 |
| **Warehouse** | wood:10, stone:5 | 12 тиков | 120 |
| **Farm** | wood:6 | 8 тиков | 60 |
| **Workshop** | wood:12, stone:8 | 15 тиков | 150 |

### Как это работает

- Поселенец **идёт к месту строительства**
- Потребляет ресурсы из инвентаря
- **Строит по 1 тику** за раз (прогресс-бар жёлтый)
- Здание появляется полупрозрачным, становится ярким по завершении

## Что на экране

- **Жёлтый круг** — поселенец «Worker»
- **Красные квадраты** — ресурсы (wood, stone)
- **Цветные квадраты** — здания (с_progress-баром при строительстве)
- **Зелёный/синий бар** — голод / энергия
- **Жёлтая линия** — маршрут движения
- **Нижняя панель** — инфо, кнопки зданий, Save/Load

## Типы клеток

| Клетка | Цвет | Проходимость | Стоимость |
|---|---|---|---|
| Трава | Зелёная | Да | 1 |
| Песок | Жёлтая | Да | 1 |
| Грунт | Коричневая | Да | 1 |
| Камень | Серый | Да | 2 |
| Вода | Синяя | **Нет** | — |

## Механика

- Мир обновляется **по тикам** (каждые 500мс)
- Каждый тик: голод, энергия, выполнение задач
- Задачи выполняются **по приоритету**
- Путь прокладывается **A*** по сетке
- Ресурсы расходуются при строительстве из инвентаря поселенца

## Пример прохождения

1. Кликни по клетке рядом с деревом — поселенец пойдёт
2. Кликни по дереву — собери ресурс (повтори несколько раз)
3. Нажми `[House]` — войди в режим строительства
4. Кликни по пустой клетке — поселенец построит дом
5. Повтори для склада, фермы, мастерской
